# X Bookmark Manager

Twitter/X のブックマークを自分で管理・検索できる個人用Webアプリです。
データはすべてブラウザ内（IndexedDB）に保存され、外部サーバーには一切送信されません。

## 特徴

- 完全無料・サーバー不要
- データは端末内にのみ保存（プライバシー重視）
- スマホ・PC両対応のレスポンシブデザイン
- GitHub Pages でそのまま公開可能

## 使い方

### Step 1: ブックマークデータを収集する

1. PCブラウザで [x.com/i/bookmarks](https://x.com/i/bookmarks) を開く
2. ブラウザの開発者ツールを開く（F12 または右クリック → 検証）
3. 「コンソール」タブを開く
4. **初回のみ**: `allow pasting` と入力して Enter を押す（下記参照）
5. `collector.js` の内容をコピーしてコンソールに貼り付け、Enterを押す
6. ページを自動スクロールしてブックマークを収集します（件数によって数十秒かかります）
7. 完了すると `bookmarks.json` が自動ダウンロードされます

> **「allow pasting」警告について**
>
> Chrome・Edge のコンソールを初めて使うと、次のような警告が表示されます：
>
> ```
> Warning: Don't paste code into the DevTools Console...
> Please type 'allow pasting' below and press Enter to allow pasting.
> ```
>
> これはブラウザのセキュリティ機能です。コンソールに `allow pasting` と入力して Enter を押すと貼り付けが有効になります。一度許可すれば次回からは不要です。

### Step 2: アプリにインポートする

1. `index.html` をブラウザで開く（またはGitHub Pagesのページにアクセス）
2. `bookmarks.json` をアプリにドラッグ＆ドロップ、またはヘッダーの「インポート」ボタンからファイルを選択する
3. ブックマークが表示されます

### 追加インポート

同じ操作を繰り返すと差分のみ追加されます（既存データは重複しません）。

## 機能（フェーズ1）

| 機能 | 説明 |
|------|------|
| ブックマーク一覧 | カード形式。ユーザー名・本文・画像・日付を表示 |
| キーワード検索 | ツイート本文・ユーザー名でリアルタイム検索 |
| 手動タグ付け | カードごとにタグを追加・削除 |
| タグ絞り込み | タグ一覧から選択してフィルタリング |
| インポート | JSON ファイルのドラッグ＆ドロップまたはファイル選択 |
| データ永続化 | IndexedDB に保存（ページを閉じてもデータが消えない） |

## ファイル構成

```
index.html      — アプリ本体（これ1つで動作します）
collector.js    — ブラウザコンソールで実行するデータ収集スクリプト
README.md       — このファイル
```

## GitHub Pages での公開

1. このリポジトリを自分の GitHub アカウントにフォーク（または新規作成）
2. Settings → Pages → Source: `main` ブランチ / `/ (root)` を選択して Save
3. 表示されたURLにアクセスするとアプリが使えます

データはあなたのブラウザ内にのみ保存されます。GitHub にアップロードされることはありません。
